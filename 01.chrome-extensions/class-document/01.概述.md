* #### Chrome插件的安装

    1. 要使用Chrome浏览器 😑
    2. 科学上网
  * 步骤：
    1. 菜单 -> 更多工具 -> 扩展程序 -> 汉堡包菜单 -> 打开Chrome应用商店
    2. 如果第一步找不到，可以直接[点击这里](https://chrome.google.com/webstore/category/extensions?hl=zh-CN)
    3. 检索我们想要查找的插件
    4. 安装
    5. 使用
    6. 卸载
